# Simple Int Stack

## Description
A small C++ project demonstrating a simple integer stack ADT using a
static array. The project tests stack operations including push, pop,
peek, overflow handling, underflow handling, full stack behavior, and
randomized testing.

## Project Structure
- main.cpp / main.h: application entry point and stack testing
- stack.h / stack.cpp: Stack class implementation using a static array

## How to Build
g++ -I ./ *.cpp

## How to Run
a.out

## Requirements
- g++ compiler
- Linux/Ubuntu environment
